Documentation
Back-end
- Install MySQL / phpMyAdmin if not available.
- Create new DB: "db-ericko"
- Run "npx sequelize-cli db:migrate"
- Run "npm start" on "back-end" folder
- Back-end will run in "localhost:3001"

Front-end
- Run "npm start" on "front-end" folder
- Front-end will run in "localhost:3000"
